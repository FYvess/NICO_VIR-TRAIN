PM Breakdown Checklist

Perform a visual inspection of all components and connections.

  • Check for loose connections
  • Inspect for physical damage
  • Verify all cables are properly seated

Test and verify all power supplies are functioning correctly.

  • Measure output voltages
  • Check for fluctuations under load
  • Inspect cooling fans

Verify all sensors are within calibration specifications.

  • Test with reference standards
  • Document readings
  • Flag any out-of-spec sensors

Run system diagnostics to verify software functionality.

  • Execute built-in test routines
  • Verify error logging
  • Check system response times
